超市后台管理

本后台系统用于对在线超市的管理，包括：

用户管理(对已注册到商城的顾客进行管理，包括更新用户等级和删除用户，但没有权限更改和查看顾客的个人信息)

商品分类管理(对商品的品类进行管理，包括增加分类，更改分类名，删除分类)

商品管理(对商品在商品分类的基础上进行增加/删除操作)
